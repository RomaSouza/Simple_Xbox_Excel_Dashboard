# 📊 Sales Dashboard (Excel)

## 📌 About the Project  
This is a sales dashboard built in Microsoft Excel, focused on making data easy to read and actually useful.

Instead of just throwing numbers on a sheet, the idea here was to organize everything in a clean and interactive way, so you can quickly understand what’s going on with the sales. Nothing crazy tho.

## ⚙️ Features  
- 📊 Pivot Tables to structure and summarize data  
- 📈 Pivot Charts for dynamic visualization  
- 🎛️ Slicers for interactive filtering  
- 📅 Time-based analysis (months, periods, etc.)  
- 💰 Key metrics like revenue, sales volume, and performance  

## 🛠️ Tools Used  
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Data Slicers  
- Basic data formatting  

## 🚀 How to Use  
1. Open the `.xlsx` file in Excel  
2. Use the slicers to filter the data  
3. Check the charts and indicators to get insights  

## 🎯 Goal  
The main goal of this project is to show skills in:
- Data organization  
- Data visualization  
- Building interactive dashboards  
- Using Excel beyond the basics  

## 📚 What I Learned  
- How to structure raw data properly  
- How to build cleaner and more useful dashboards  
- How to make data easier to understand

# 📊 Dashboard de Vendas (Excel)

## 📌 Sobre o Projeto  
Esse projeto é um dashboard de vendas feito no Excel, com foco em deixar os dados mais claros e fáceis de entender.

A ideia não foi só jogar números na planilha, mas organizar tudo de um jeito visual e interativo pra facilitar a análise.

## ⚙️ Funcionalidades  
- 📊 Tabelas dinâmicas para organizar os dados  
- 📈 Gráficos dinâmicos para visualização  
- 🎛️ Segmentação de dados (slicers) para filtros interativos  
- 📅 Análise por período (meses, datas, etc.)  
- 💰 Métricas como faturamento, volume de vendas e desempenho  

## 🛠️ Ferramentas Utilizadas  
- Microsoft Excel  
- Tabelas Dinâmicas  
- Gráficos Dinâmicos  
- Segmentação de Dados  
- Formatação básica  

## 🚀 Como Usar  
1. Abra o arquivo `.xlsx` no Excel  
2. Use os filtros (segmentações) para explorar os dados  
3. Analise os gráficos e indicadores  

## 🎯 Objetivo  
O objetivo desse projeto é mostrar habilidades em:
- Organização de dados  
- Visualização de dados  
- Criação de dashboards interativos  
- Uso do Excel além do básico  

## 📚 Aprendizados  
- Melhor estruturação de dados  
- Criação de dashboards mais limpos e úteis  
- Como deixar a informação mais fácil de entender
